# Ghost-Note

Site web de Ghost-Note, activité de transmission musicale à Toulouse.

## À propos

Ghost-Note est une structure de transmission musicale organique, vivante et informelle qui propose une alternative aux parcours académiques classiques. Le site présente les différents formats d'accompagnement (cours individuels, cours collectifs CDMC, ateliers), les thématiques abordées (Piano, Théorie Musicale, Rythmique, MAO) et les événements à venir.

## Technologies

- **React** avec TypeScript
- **Vite** pour le build et le développement
- **React Router** (HashRouter) pour la navigation
- **Tailwind CSS** pour le styling
- Optimisé pour le SEO, l'accessibilité et les performances

## Développement

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Build de production
npm run build
```

## Structure

- `/pages` - Pages principales du site
- `/components` - Composants réutilisables
- `/public` - Assets statiques (images, logos, favicon)
